# llm — the two-library build

This workspace holds the plan and build log for `ember` (GPT model + training
library) and `forge` (GPU kernel-generation library), which are separate git
repos sitting alongside this file.

- [PLAN.md](PLAN.md) — the full plan: milestones, metrics, learning system, cadence.
- [LOG.md](LOG.md) — the build log. One entry per week minimum (Monday review).
- `ember/` — the model library. **Start here: `ember/docs/milestones/A0.md`.**
- `forge/` — the kernel library. Starts at B0, after ember A1. Deliberately empty until then.
